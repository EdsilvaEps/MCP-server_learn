# Project: goose-logs-mcp
Instructions for AI Agents.

## 🛠 Development Workflow
- **Environment**: Avoid installing packages globally; always use a virtual environment (`venv`).
- **Tooling**: Use `python` or `uv` for development and package management.
- **Documentation**: Modify the `README.md` whenever significant code changes are made to keep documentation in sync.

## 🌐 Capabilities & Tooling
- **Webscraping**: If available, use the `firecrawl` MCP server for webscraping tasks.
